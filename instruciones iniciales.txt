Ficha Técnica: Plantilla de Vistas de Contenido
1. Tecnologías y Librerías Base
Framework: React (Componentes funcionales, hooks como useState).
Estilos: Tailwind CSS.
Iconografía: lucide-react.
Animaciones: Clases de Tailwind y tailwindcss-animate (ej. animate-in, fade-in, slide-in-from-bottom).
2. Paleta de Colores y Tipografía (Clases Tailwind)
El diseño se basa en un tema oscuro (Dark Mode) con acentos de un color primario y uso de transparencias para el efecto cristal.

Fondos base: bg-slate-900, bg-black/30, transparentes.
Textos principales (Títulos): text-white.
Textos secundarios (Párrafos): text-slate-300, text-slate-400.
Acentos y Destacados: Usa el color primary. Clases típicas: text-primary-400, text-primary-500, bg-primary-500/10 (para fondos suaves de cajas destacadas).
Tipografía: Variaciones de peso extremas para contraste. Títulos principales en font-black (muy grueso), títulos de sección en font-bold. Tarjetas con subtítulos en font-medium o en mayúsculas uppercase tracking-widest text-sm.
3. Disposición y Estructura Principal del Contenedor
Todo el componente debe estar envuelto en un contenedor principal con animación de entrada y control de espaciado:

jsx
<div className="w-full space-y-12 animate-in fade-in slide-in-from-bottom-4 duration-700 pb-20">
    {/* Contenido */}
</div>
4. Patrones UI (Lo que debes replicar)
A. Cabecera (Header con Botón Volver)
Utiliza un botón de retroceso circular con efecto cristal, un supra-título (unidad/tema) y el título principal de la vista.

jsx
<div className="flex items-center gap-4">
    <button onClick={onBack} className="p-2 rounded-full glass-morphism text-slate-400 hover:text-primary-400 hover:scale-110 transition-all cursor-pointer">
        <ArrowLeft className="w-6 h-6" />
    </button>
    <div>
        <p className="text-primary-500 font-bold text-sm uppercase tracking-widest mb-1 flex items-center gap-2">
            <span className="w-8 h-[1px] bg-primary-500"></span>
            Categoría o Unidad
        </p>
        <h1 className="text-4xl md:text-5xl font-black text-white tracking-tight">
            Título Principal de la Vista
        </h1>
    </div>
</div>
B. Sección de Contenido Normal (Glassmorphism)
Cada bloque de información importante está contenido en una sección que usa el efecto cristal, bordes redondeados pronunciados y un buen relleno (padding).

jsx
<section className="glass-morphism rounded-3xl p-8 space-y-8">
    <div className="grid grid-cols-1 md:grid-cols-2 gap-8 items-start">
        {/* Columna Izquierda: Teoría */}
        <div className="space-y-6">
            <h2 className="text-2xl font-bold text-white">Título de Sección</h2>
            <p className="text-slate-300 leading-relaxed text-lg">Texto descriptivo...</p>
        </div>
        
        {/* Columna Derecha: Tarjeta destacada o lista de conceptos */}
        <div className="bg-slate-900/60 p-6 rounded-2xl border border-white/5 h-full">
            <h3 className="text-white font-bold flex items-center gap-2">
               <Target className="w-5 h-5 text-primary-400" /> Título de Tarjeta
            </h3>
            {/* Contenido de tarjeta */}
        </div>
    </div>
</section>
C. Tarjetas de Componentes o Conceptos (Estilo "Grid")
Usado para diccionarios, listados de características o ejemplos. Son cajas oscuras con bordes sutiles.

jsx
<div className="bg-slate-900/60 p-6 rounded-3xl border border-white/5 flex flex-col">
    <h3 className="text-xl font-bold text-white mb-2 flex items-center gap-2">
        <span className="text-primary-500/50 font-black">1.</span> Título del Item
    </h3>
    <p className="text-sm text-slate-300 mb-4">Descripción del item.</p>
    
    {/* Tags de colores */}
    <div className="flex flex-wrap gap-2 mb-4">
        <span className="bg-white/5 text-slate-300 px-2 py-1 rounded text-xs border border-white/10">Etiqueta 1</span>
        <span className="bg-primary-500/10 text-primary-300 px-2 py-1 rounded text-xs border border-primary-500/20">Etiqueta 2</span>
    </div>
</div>
D. Componente Marcador de Imagen (

ImagePlaceholder
)
Si vas a armar una vista nueva y aún no tienes las imágenes, usa exactamente este subcomponente arriba del JSX:

jsx
const ImagePlaceholder = ({ title, height = "h-48" }) => (
    <div className={`w-full ${height} bg-slate-900/60 border-2 border-dashed border-primary-500/30 rounded-2xl flex flex-col items-center justify-center text-slate-500 hover:border-primary-500/60 hover:text-primary-400 hover:bg-slate-900/80 transition-all cursor-pointer group mt-4 mb-2`}>
        <ImageIcon className="w-8 h-8 mb-2 group-hover:scale-110 transition-transform" />
        <p className="text-sm font-medium uppercase tracking-widest">{title}</p>
        <p className="text-xs mt-1 opacity-60">(Espacio para imagen)</p>
    </div>
);
E. Sección Acordeón / Colapsable (Para contenido extenso)
Estructura interactiva y animada para ocultar contenido pesado. Ojo: Requiere usar un estado (useState) para controlar si está abierto o cerrado.

jsx
<section className="space-y-4">
    <button onClick={() => setIsOpen(!isOpen)} className="w-full flex items-center justify-between p-6 glass-morphism rounded-2xl hover:bg-white/5 transition-all outline-none cursor-pointer group">
        <div className="flex items-center gap-4">
            <div className={`p-3 rounded-xl transition-colors duration-300 ${isOpen ? 'bg-primary-500 text-white' : 'bg-white/5 text-primary-400'}`}>
                <BookOpen className="w-6 h-6" /> {/* Icono referente al contenido */}
            </div>
            <div className="text-left">
                <h2 className="text-2xl font-bold text-white group-hover:text-primary-400 transition-colors">Título del Desplegable</h2>
            </div>
        </div>
        <div className={`p-2 rounded-full ${isOpen ? 'bg-primary-500/10 text-primary-400' : 'bg-white/5 text-slate-400'}`}>
            {isOpen ? <ChevronDown className="w-6 h-6" /> : <ChevronRight className="w-6 h-6" />}
        </div>
    </button>
    <div className={`transition-all duration-700 ease-in-out overflow-hidden ${isOpen ? 'opacity-100 max-h-[5000px] mt-4' : 'opacity-0 max-h-0'}`}>
        <div className="glass-morphism rounded-[2.5rem] p-8 md:p-14 border border-primary-500/10">
            {/* Contenido oculto */}
        </div>
    </div>
</section>
